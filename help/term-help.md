ctrl + a - в начало
ctrl + e - в конец
ctrl + k - удалить после курсура
ctrl + u - удалить до курсура
ctrl + w - удалить слово перед курсором
ctrl + y - вставить последнее из буфера
ctrl + d - закрыть окно терминала
ctrl + z - отправить процесс в бэкграунд (jobs, fg)
ctrl + xx - выделить последнее слово
alt + b - на слово назад
alt + f - на слово вперед
alt + c - перевести выделенную букву в верхний регистр
alt + t - свапнуть слово с предыдущим
