#navigation

- Move tab forward (to the right): Press Ctrl + Shift + . (period).
- Move tab backward (to the left): Press Ctrl + Shift + , (comma).
- New split window (auto-layout) Ctrl + Shift + Enter
- Close current window Ctrl + Shift + W
- Next window (focus) Ctrl + Shift + ]
- Previous window (focus) Ctrl + Shift + [
- Change layout (cycle through available layouts) Ctrl + Shift + L
